# 🍉 合成大西瓜 - Watermelon Merge Game

一个精美、流畅的合成大西瓜游戏，支持移动端和桌面端。

## ✨ 特性

- 🎨 **精美画面**：渐变背景、圆角设计、阴影效果
- 🎮 **流畅操作**：点击或空格键掉落水果，支持触屏
- 🌈 **10个等级**：从葡萄到西瓜，逐步合成
- 💎 **粒子特效**：水果合并时的爆炸效果
- 📊 **分数系统**：实时分数和最高分记录
- 📱 **响应式设计**：完美适配手机和电脑
- ⏸️ **暂停功能**：随时暂停/继续游戏
- 🎯 **危险线提示**：清晰的游戏结束判定线

## 🎮 游戏规则

1. 点击屏幕或按空格键掉落水果
2. 相同水果碰撞会合成更大的水果
3. 水果超过顶部危险线游戏结束
4. 合成大西瓜获得最高分 512 分！
